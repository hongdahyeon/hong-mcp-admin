# AI Agent rules

## 1. 항상 답변은 한국말로 진행한다.
## 2. 항상 코드를 반영하기 전, 질문 후 응답 이후에 진행한다.
## 3. 항상 수행 전에 implementation_plan.md 파일에 작업을 정리하고 진행한다.
- implementation_plan.md 파일은 항상 누적해서 적는다.
## 4. 항상 수행 전에 todo.md 파일을 적어두고, 수행 과정에 따라 체크 표시를 하며 진행한다.
- todo.md 파일은 항상 누적해서 적는다.
## 5. backend/ 하위 파일은 절대로 건드리지 않는다.
